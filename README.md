# S5 Truck Repair Landing Page

Static marketing page for S5 Truck Repair that follows the provided “perfect landing page” structure:

## Features

- Hero section with bold H1, click-to-call buttons, trust badges, and sticky header.
- Intro, services grid, coverage map, benefit highlights, photo gallery, reviews, contact form, and SEO-friendly footer.
- Yellow/black/red palette, responsive layout, and sticky call button for mobile.

## Local setup

1. Open `index.html` in a browser or serve it from any static file host.
2. `styles.css` already linked – no build step required.
3. Update the map embed or phone/email copy as needed before publishing.

## Deployment

Push the repository to any static-hosting platform (GitHub Pages, Netlify, Vercel, etc.).
